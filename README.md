# Home

All of my configuration in one handy place. 

The goal is to Quickly and easily get a reproducible development setup on a new Unix machine. To that end, I am using the [Nix Package Manager](https://nixos.org/) for all package management and attempting to centralize all configuration using the `$XDG_CONFIG_HOME`environment variable following the [XDG Base Directory Specification](https://wiki.archlinux.org/index.php/XDG_Base_Directory).
